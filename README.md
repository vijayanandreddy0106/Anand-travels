# Anand Travels Website

## 🌍 Project Overview
Anand Travels is a responsive travel website created using HTML, CSS, and JavaScript. It allows users to explore travel destinations, view trip details, browse the gallery, and book trips through a contact form.

## 🚀 Features
- Home Page
- About Us Page
- Our Trips Page
- Gallery Page
- Contact Us Page
- Responsive Design
- JavaScript Form Validation
- Simple and User-Friendly Interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
anand-travels/
│── index.html
│── about.html
│── packages.html
│── gallery.html
│── contact.html
│── style.css
│── script.js
│── README.md
```

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server.
4. Explore the website.
